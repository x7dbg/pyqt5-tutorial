# 菜单和工具栏

打开任何一个桌面软件——Word、浏览器、记事本——你都会看到这三样东西：
- 顶部的**菜单栏**（文件、编辑、视图……）
- 下面的**工具栏**（保存、撤销、复制……）
- 底部的**状态栏**（显示一些提示信息）

这就是桌面应用的"标准装修"。本章就来学习怎么给你的程序也装上这三件套。

---

## 1. 先认识 QMainWindow

前面我们用的都是 `QWidget`，它是最基础的窗口。但如果你想做正经的桌面软件，得用 `QMainWindow`。

`QMainWindow` 就像一个**已经装修好的毛坯房**，自带了这些区域：

```
┌─────────────────────────────────┐
│  菜单栏（文件 编辑 视图 帮助）    │
├─────────────────────────────────┤
│  工具栏（💾 🔄 ✂️ 📋）           │
├─────────────────────────────────┤
│                                 │
│                                 │
│        中央区域（你放内容的地方）  │
│                                 │
│                                 │
├─────────────────────────────────┤
│  状态栏（准备就绪）               │
└─────────────────────────────────┘
```

### 1.1 最简单的 QMainWindow

```python
# -*- coding: utf-8 -*-

import sys
from PyQt5.QtWidgets import QMainWindow, QApplication


class Example(QMainWindow):

    def __init__(self):
        super().__init__()

        self.initUI()


    def initUI(self):               

        self.statusBar().showMessage('准备就绪')

        self.setGeometry(300, 300, 250, 150)
        self.setWindowTitle('状态栏')    
        self.show()


if __name__ == '__main__':

    app = QApplication(sys.argv)
    ex = Example()
    sys.exit(app.exec_())
```

运行后你会看到一个窗口，底部有一条状态栏，上面写着"准备就绪"。

### 1.2 核心代码

```python
self.statusBar().showMessage('准备就绪')
```

就这一行，状态栏就搞定了。

`statusBar()` 这个方法很聪明：
- 第一次调用 → 自动创建状态栏
- 后面再调用 → 直接返回已经创建好的状态栏

你不用管它怎么创建的，拿来用就行。

> 💡 **小贴士**：`QMainWindow` 和 `QWidget` 的区别？`QWidget` 是毛坯房，`QMainWindow` 是精装房。想做正经软件就用 `QMainWindow`。

## 2. 简单菜单

### 2.1 创建第一个菜单

每个软件都有"文件"菜单，我们也来做一个。

```python
# -*- coding: utf-8 -*-

import sys
from PyQt5.QtWidgets import QMainWindow, QAction, qApp, QApplication
from PyQt5.QtGui import QIcon


class Example(QMainWindow):

    def __init__(self):
        super().__init__()

        self.initUI()


    def initUI(self):               

        exitAct = QAction(QIcon('exit.png'), '退出(&X)', self)        
        exitAct.setShortcut('Ctrl+Q')
        exitAct.setStatusTip('退出应用')
        exitAct.triggered.connect(qApp.quit)

        self.statusBar()

        menubar = self.menuBar()
        fileMenu = menubar.addMenu('文件(&F)')
        fileMenu.addAction(exitAct)

        self.setGeometry(300, 300, 300, 200)
        self.setWindowTitle('简单菜单')    
        self.show()


if __name__ == '__main__':

    app = QApplication(sys.argv)
    ex = Example()
    sys.exit(app.exec_())
```

运行后，窗口顶部会出现"文件"菜单，点开后有一个"退出"选项，点击就关闭程序。

### 2.2 拆解一下

**第一步：创建动作（QAction）**

```python
exitAct = QAction(QIcon('exit.png'), '退出(&X)', self)        
exitAct.setShortcut('Ctrl+Q')
exitAct.setStatusTip('退出应用')
exitAct.triggered.connect(qApp.quit)
```

`QAction` 是菜单和工具栏的"核心零件"。一个动作可以：
- 放在菜单里 → 变成菜单项
- 放在工具栏里 → 变成工具按钮
- 绑定快捷键 → 键盘也能触发

这就像你写了一封信，可以寄出去、可以贴墙上、可以发朋友圈，内容是一样的，只是展示方式不同。

这段代码做了四件事：
1. 创建动作，文字是"退出"，带个图标
2. 绑定快捷键 `Ctrl+Q`
3. 鼠标悬停时，状态栏显示"退出应用"
4. 点击后执行 `qApp.quit`（退出程序）

> 💡 **关于 `(&X)`**：那个 `&X` 是助记符（mnemonic），当"文件"菜单已经打开时，按 `X` 键就能激活这个菜单项。`&F` 同理，按 `Alt+F` 可以打开"文件"菜单。这是 Windows 的传统。

**第二步：创建菜单**

```python
menubar = self.menuBar()
fileMenu = menubar.addMenu('文件(&F)')
fileMenu.addAction(exitAct)
```

三行代码：
1. 拿到菜单栏
2. 添加一个"文件"菜单
3. 把"退出"动作放进去

> 🎮 **动手试试**：把 `qApp.quit` 改成 `self.close`，效果一样吗？

## 3. 子菜单

### 3.1 什么是子菜单？

就是菜单里套菜单，鼠标悬停会弹出下一级菜单。

```
文件
├── 新建
└── 导入
    ├── 导入邮件
    ├── 导入文件
    └── 导入图片
```

### 3.2 代码实现

```python
# -*- coding: utf-8 -*-

import sys
from PyQt5.QtWidgets import QMainWindow, QAction, QMenu, QApplication


class Example(QMainWindow):

    def __init__(self):
        super().__init__()

        self.initUI()


    def initUI(self):         

        menubar = self.menuBar()
        fileMenu = menubar.addMenu('文件')
        
        # 创建子菜单
        impMenu = QMenu('导入', self)
        impAct = QAction('导入邮件', self) 
        impMenu.addAction(impAct)

        # 普通菜单项
        newAct = QAction('新建', self) 
        fileMenu.addAction(newAct)
        
        # 把子菜单挂到文件菜单下
        fileMenu.addMenu(impMenu)

        self.setGeometry(300, 300, 300, 200)
        self.setWindowTitle('子菜单')    
        self.show()


if __name__ == '__main__':

    app = QApplication(sys.argv)
    ex = Example()
    sys.exit(app.exec_())
```

关键就一行：

```python
fileMenu.addMenu(impMenu)
```

把"导入"子菜单挂到"文件"菜单下面。

> 💡 **理解**：`addMenu()` 是挂子菜单，`addAction()` 是加菜单项。别搞混了。

## 4. 勾选菜单

### 4.1 什么是勾选菜单？

就是菜单项前面有个 ✅ 或 ☐，可以切换开关状态。比如 Word 里的"显示标尺"、"显示网格"。

### 4.2 代码实现

```python
# -*- coding: utf-8 -*-

import sys
from PyQt5.QtWidgets import QMainWindow, QAction, QApplication


class Example(QMainWindow):

    def __init__(self):
        super().__init__()

        self.initUI()


    def initUI(self):         

        self.statusbar = self.statusBar()
        self.statusbar.showMessage('准备就绪')

        menubar = self.menuBar()
        viewMenu = menubar.addMenu('视图')

        # 创建可勾选的菜单项
        viewStatAct = QAction('查看状态栏', self, checkable=True)
        viewStatAct.setStatusTip('查看状态栏')
        viewStatAct.setChecked(True)  # 默认勾选
        viewStatAct.triggered.connect(self.toggleMenu)

        viewMenu.addAction(viewStatAct)

        self.setGeometry(300, 300, 300, 200)
        self.setWindowTitle('勾选菜单')    
        self.show()


    def toggleMenu(self, state):
        """根据勾选状态显示/隐藏状态栏"""
        if state:
            self.statusbar.show()
        else:
            self.statusbar.hide()


if __name__ == '__main__':

    app = QApplication(sys.argv)
    ex = Example()
    sys.exit(app.exec_())
```

### 4.3 核心代码

```python
viewStatAct = QAction('查看状态栏', self, checkable=True)
```

`checkable=True` 让这个菜单项变成可勾选的。

```python
viewStatAct.setChecked(True)  # 默认勾选
```

设置初始状态为勾选。

```python
def toggleMenu(self, state):
    if state:
        self.statusbar.show()
    else:
        self.statusbar.hide()
```

`state` 参数就是勾选状态：`True` = 勾选，`False` = 未勾选。根据状态来决定显示还是隐藏状态栏。

## 5. 右键菜单

### 5.1 什么是右键菜单？

右键点击时弹出的菜单，也叫上下文菜单。不同地方右键，出现的菜单可能不一样。

### 5.2 代码实现

```python
# -*- coding: utf-8 -*-

import sys
from PyQt5.QtWidgets import QMainWindow, qApp, QMenu, QApplication


class Example(QMainWindow):

    def __init__(self):
        super().__init__()

        self.initUI()


    def initUI(self):         

        self.setGeometry(300, 300, 300, 200)
        self.setWindowTitle('右键菜单')    
        self.show()


    def contextMenuEvent(self, event):
        """右键点击时触发"""
        cmenu = QMenu(self)

        newAct = cmenu.addAction("新建")
        opnAct = cmenu.addAction("打开")
        quitAct = cmenu.addAction("退出")
        
        # 在鼠标位置弹出菜单
        action = cmenu.exec_(self.mapToGlobal(event.pos()))

        # 判断用户点了哪个
        if action == quitAct:
            qApp.quit()


if __name__ == '__main__':

    app = QApplication(sys.argv)
    ex = Example()
    sys.exit(app.exec_())
```

### 5.3 核心代码

```python
def contextMenuEvent(self, event):
```

重写这个方法，右键时就会触发。

```python
action = cmenu.exec_(self.mapToGlobal(event.pos()))
```

这行做了两件事：
1. 在鼠标位置弹出菜单
2. 返回用户点了哪个菜单项

```python
if action == quitAct:
    qApp.quit()
```

判断用户点的是不是"退出"，是的话就关闭程序。

> 🎮 **动手试试**：在窗口里右键点击，看看菜单弹出来没有。

## 6. 工具栏

### 6.1 工具栏 vs 菜单

| 对比 | 菜单 | 工具栏 |
|------|------|--------|
| 位置 | 顶部 | 通常在菜单下面 |
| 内容 | 所有功能 | 常用功能 |
| 操作 | 要点两下（菜单→菜单项） | 点一下就行 |
| 图标 | 可有可无 | 一般都有 |

工具栏就是菜单的"快捷方式"，把最常用的功能拿出来放一排。

### 6.2 代码实现

```python
# -*- coding: utf-8 -*-

import sys
from PyQt5.QtWidgets import QMainWindow, QAction, qApp, QApplication
from PyQt5.QtGui import QIcon


class Example(QMainWindow):

    def __init__(self):
        super().__init__()

        self.initUI()


    def initUI(self):               

        exitAct = QAction(QIcon('exit.png'), '退出', self)
        exitAct.setShortcut('Ctrl+Q')
        exitAct.triggered.connect(qApp.quit)

        # 创建工具栏
        self.toolbar = self.addToolBar('退出')
        self.toolbar.addAction(exitAct)

        self.setGeometry(300, 300, 300, 200)
        self.setWindowTitle('工具栏')    
        self.show()


if __name__ == '__main__':

    app = QApplication(sys.argv)
    ex = Example()
    sys.exit(app.exec_())
```

### 6.3 核心代码

```python
self.toolbar = self.addToolBar('退出')
self.toolbar.addAction(exitAct)
```

两行搞定：
1. 创建工具栏
2. 把动作加进去

注意：`exitAct` 和菜单里用的是同一个！这就是 `QAction` 的好处——一次创建，多处使用。

## 7. 完整的主窗口

### 7.1 把前面的都组合起来

现在我们把菜单栏、工具栏、状态栏、中央控件全部组合在一起，做一个完整的主窗口。

```python
# -*- coding: utf-8 -*-

import sys
from PyQt5.QtWidgets import QMainWindow, QTextEdit, QAction, QApplication
from PyQt5.QtGui import QIcon


class Example(QMainWindow):

    def __init__(self):
        super().__init__()

        self.initUI()


    def initUI(self):               

        # 1. 中央区域：文本编辑器
        textEdit = QTextEdit()
        self.setCentralWidget(textEdit)

        # 2. 创建退出动作
        exitAct = QAction(QIcon('exit.png'), '退出', self)
        exitAct.setShortcut('Ctrl+Q')
        exitAct.setStatusTip('退出应用')
        exitAct.triggered.connect(self.close)

        # 3. 状态栏
        self.statusBar()

        # 4. 菜单栏
        menubar = self.menuBar()
        fileMenu = menubar.addMenu('文件')
        fileMenu.addAction(exitAct)

        # 5. 工具栏
        toolbar = self.addToolBar('退出')
        toolbar.addAction(exitAct)

        self.setGeometry(300, 300, 350, 250)
        self.setWindowTitle('主窗口')    
        self.show()


if __name__ == '__main__':

    app = QApplication(sys.argv)
    ex = Example()
    sys.exit(app.exec_())
```

### 7.2 结构拆解

```
┌──────────────────────────────────┐
│  文件                              │ ← 菜单栏
├──────────────────────────────────┤
│  [退出]                           │ ← 工具栏
├──────────────────────────────────┤
│                                  │
│                                  │
│      文本编辑器（中央区域）         │
│                                  │
│                                  │
├──────────────────────────────────┤
│  退出应用                          │ ← 状态栏
└──────────────────────────────────┘
```

这个结构就是大多数桌面软件的样子。以后你做项目，基本也是这个框架。

> 📌 **记住**：`setCentralWidget()` 只能设置一个中央控件。如果需要放多个控件，先把它们用布局组合成一个 widget，再设为中央控件。
