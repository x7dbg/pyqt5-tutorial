# PyQt5 完整教程

> 从零开始，用 Python 打造桌面应用程序

一套完整的 PyQt5 学习教程，从环境搭建到项目实战，涵盖桌面开发的核心知识点。所有代码示例均已中文化，注释详细，开箱即用。

---

## 💬 作者的话

我当初学 PyQt5 的时候，踩过不少坑。网上的教程要么太老（Python 2 时代的），要么太浅（只教你画个窗口就没了），要么太深（一上来就讲 Model/View 架构，新人根本看不懂）。

所以我决定整理一套**从零开始、能跑、能懂**的教程。

这套教程的代码示例参考了 [zetcode PyQt5 教程](https://zetcode.com/gui/pyqt5/)（一个非常经典的 PyQt5 入门资源），但所有代码我都重新写了注释，文字内容是自己整理和编写的。如果你发现某些例子和 zetcode 很像，那不是巧合——我确实参考了它，但讲解方式是我自己的。

我的目标很简单：**让你看完就能用，不用再去翻官方文档**。

如果你在学习过程中遇到问题，欢迎反馈，我会尽量更新教程来覆盖这些坑。

## 🎯 适合谁学？

- **Python 初学者**：已经会基础语法，想学做图形界面
- **想转桌面开发的 Web 开发者**：了解 PyQt5 的组件和布局思路
- **需要做内部工具的后端工程师**：快速搭建管理界面、配置工具
- **学生/老师**：课程设计、毕业设计需要 GUI 界面

**前置要求**：了解 Python 基础（变量、函数、类、模块），不需要有 GUI 开发经验。

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install PyQt5
pip install pyqt5-tools
```

### 2. 运行第一个示例

创建 `hello.py`，复制以下代码：

```python
# -*- coding: utf-8 -*-
import sys
from PyQt5.QtWidgets import QApplication, QWidget

if __name__ == '__main__':
    app = QApplication(sys.argv)
    w = QWidget()
    w.resize(250, 150)
    w.move(300, 300)
    w.setWindowTitle('我的第一个窗口')
    w.show()
    sys.exit(app.exec_())
```

运行：

```bash
python hello.py
```

你会看到一个窗口弹出来！恭喜，你已经迈出了第一步。

### 3. 开始学习

从 [第1章-环境搭建](01-环境搭建.md) 开始，按顺序学习。
